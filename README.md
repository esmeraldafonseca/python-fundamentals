# Python Fundamentals

This repository contains my Python learning journey, including exercises, experiments, and small projects created while studying Python from different resources and through personal practice.

Although many exercises are based on **Curso em Vídeo** by **Gustavo Guanabara**, the repository has expanded to include additional topics such as Object-Oriented Programming, JSON, SQLite3, and desktop application development with CustomTkinter.

## Topics Covered

* Python Basics
* Arithmetic Operators
* Modules and Imports
* String Manipulation
* Conditional Statements
* Nested Conditions
* `for` Loops
* `while` Loops
* Tuples
* Lists
* Dictionaries
* Functions
* Modules and Packages
* Error Handling
* Object-Oriented Programming (OOP)
* JSON
* SQLite3
* CustomTkinter GUI Development
