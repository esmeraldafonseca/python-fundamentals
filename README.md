# Python Fundamentals

This repository contains Python exercises and small projects based on the **"Curso em Vídeo" Python course by Gustavo Guanabara**.  
All exercises were **coded, organized, and improved by Esmeralda Fonseca**.

## Topics Covered

- Variables and Data Types  
- Conditional Statements (`if`, `else`, `elif`)  
- Loops (`for`, `while`)  
- Functions  
- Lists, Tuples, and Dictionaries  
- Modules and Packages  
- Error Handling  
- Input/Output and Formatting

## Repository Structure

```text
python-fundamentals/
│

## Goal

The goal of this repository is to practice problem solving and build a strong foundation in Python programming.
